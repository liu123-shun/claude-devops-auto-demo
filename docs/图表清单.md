## 图 4.1 JWT 登录鉴权流程图

```
┌─────────┐
│  开始    │
└────┬────┘
     │
     ▼
┌─────────────────────────────┐
│  用户访问登录页面              │
│  /frontend/login.html        │
└────────────┬────────────────┘
             │
             ▼
┌─────────────────────────────┐
│  GET /api/auth/captcha      │
│  获取图形验证码               │
│  (Pillow 生成4位字符+干扰线)  │
│  返回 captcha_token(JWT加密) │
│       + image(base64 PNG)   │
└────────────┬────────────────┘
             │
             ▼
┌─────────────────────────────┐
│  用户输入:                    │
│  账号 + 密码 + 验证码 + 角色  │
└────────────┬────────────────┘
             │
             ▼
     ┌───────────────┐
     │ 验证码是否正确?  │──── 否 ──► 返回 400 "验证码错误"
     └───────┬───────┘
             │ 是
             ▼
     ┌───────────────────┐
     │ sys_user 表中        │
     │ 查询 username+role  │
     │ 是否有匹配记录?      │──── 否 ──► 返回 401 "账号密码错误"
     └───────┬───────────┘
             │ 是
             ▼
     ┌───────────────────┐
     │ bcrypt.checkpw()  │
     │ 明文密码 vs 哈希值   │──── 否 ──► 返回 401 "账号密码错误"
     └───────┬───────────┘
             │ 是
             ▼
┌─────────────────────────────┐
│  登录成功                     │
│  ① 写入 login_log 表          │
│     (user_id, login_ip,       │
│      login_time, login_role)  │
│  ② 签发 JWT Token (2h有效期)  │
│     payload: user_id,         │
│     username, role, exp       │
└────────────┬────────────────┘
             │
             ▼
┌─────────────────────────────┐
│  返回 Token + 用户信息        │
│  → 前端 localStorage 存储    │
│  → 根据 role 跳转对应首页     │

│    student → student_index  │
└────────────┬────────────────┘
             │
             ▼
┌──────────────────────────────────┐
│       全局鉴权 (每个后续请求)       │
│  HTTPBearer 从 Header 取 Token    │
│  → decode_jwt_token() 解析        │
│  → 校验 role 是否匹配接口权限      │
│  → 校验数据所有权                  │
└──────────────────────────────────┘
             │
             ▼
┌─────────┐
│  结束    │
└─────────┘
```

## 图 4.3 协同过滤推荐算法流程图

```
┌──────────────┐
│    开始       │
│  用户请求推荐  │
└──────┬───────┘
       │
       ▼
┌──────────────────────────────┐
│  Step 1: 获取用户借阅历史      │
│  SELECT book_id              │
│  FROM borrow_record          │
│  WHERE reader_id = user_A    │
│  → borrowed_books = [b1,b2..]│
└──────────────┬───────────────┘
               │
     ┌─────────▼─────────┐
     │ 借阅列表是否为空?    │
     └───┬───────┬───────┘
         │ 是    │ 否
         ▼       ▼
┌────────────┐ ┌──────────────────────────────────┐
│ 冷启动处理  │ │ Step 2: 发现相似用户               │
│ 返回热门推荐│ │ SELECT DISTINCT reader_id         │
│ 结束       │ │ FROM borrow_record                │
└────────────┘ │ WHERE book_id IN borrowed_books   │
               │   AND reader_id != user_A         │
               │ → similar_users = [u3, u5, u7...]│
               └──────────────┬───────────────────┘
                              │
                ┌─────────────▼─────────────┐
                │ 相似用户列表是否为空?         │
                └─────┬───────────┬─────────┘
                      │ 是        │ 否
                      ▼           ▼
            ┌────────────┐ ┌──────────────────────────────┐
            │ 退化为分类  │ │ Step 3: 关联书籍统计            │
            │ 热门推荐    │ │ SELECT book_id, COUNT(*) AS cnt│
            │ 结束       │ │ FROM borrow_record             │
            └────────────┘ │ WHERE reader_id IN similar_ids │
                           │   AND book_id NOT IN           │
                           │       borrowed_books           │
                           │ GROUP BY book_id               │
                           │ ORDER BY cnt DESC              │
                           └──────────────┬─────────────────┘
                                          │
                                          ▼
                           ┌──────────────────────────────┐
                           │ Step 4: 结果数量不足( <3 )?    │
                           └──┬──────────────┬────────────┘
                              │ 是           │ 否
                              ▼              ▼
                   ┌────────────┐ ┌──────────────────────┐
                   │ 同分类热门  │ │ Step 5: 排序输出Top N │
                   │ 图书补充    │ │ 按 cnt 降序 +         │
                   │            │ │ stock > 0 过滤       │
                   └─────┬──────┘ └──────────┬───────────┘
                         │                   │
                         └──────┬────────────┘
                                │
                                ▼
                    ┌───────────────────┐
                    │   返回推荐结果      │
                    │   items: [book...] │
                    │   reason: 推荐理由  │
                    └─────────┬─────────┘
                              │
                              ▼
                    ┌──────────────┐
                    │     结束      │
                    └──────────────┘
```
